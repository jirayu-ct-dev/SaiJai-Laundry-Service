# Setup

```

bun create nuxt
bun install tailwindcss @tailwindcss/vite

bun install dotenv uuid

bun install better-auth
bun install --save-dev @types/node

bun install prisma
bun install prisma
bun install @prisma/client
bun prisma init

bun prisma db push
bunx @better-auth/cli@latest generate

bun install --save @line/liff

```
